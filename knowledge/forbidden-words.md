# 保健品小红书文案违禁词与敏感词清单

> 本清单基于《中华人民共和国广告法》《保健食品广告审查暂行规定》《网络食品安全违法行为查处办法》等法规整理。生成文案时必须逐条对照，确保合规。

---

## 一、绝对禁止使用的词语

### 1. 绝对化用语（广告法第九条）

最、最佳、最优、最好、最强、第一、唯一、首个、顶级、极品、绝佳、绝无仅有、独一无二、万能、100%、零风险、绝对、务必

### 2. 医疗术语与疗效宣称

治疗、治愈、药到病除、根治、痊愈、处方、临床验证、诊断、药效、疗效、用药、服药、患者、病理、手术、住院、门诊

### 3. 功效承诺类

X天见效、立竿见影、包治、保证有效、一用就好、无效退款、永不复发、彻底解决、药到病除、瞬间、立刻见效、当天见效、一周见效、百分之百

### 4. 权威背书类

医生推荐、专家认证、医院推荐、国家认证、权威机构推荐、院士推荐、诺贝尔奖、国家级、特供、专供

### 5. 与药品/医疗器械关联

替代药物、替代药品、比药还好、不用吃药、不用手术、不用去医院

## 二、保健品专项禁用表达

### 疾病名称（不得出现）

癌症、肿瘤、糖尿病、高血压、心脏病、冠心病、中风、肝炎、肝硬化、肾病、关节炎、骨质疏松、抑郁症、失眠症、甲亢、甲减、痛风、结石

> 替代写法：用"关心XX健康""关注XX问题""为XX加一道防线"等表述

### 功效宣称（不得出现）

| 禁用表达 | 说明 | 合规替代 |
|---------|------|---------|
| 降低血糖 | 暗示治疗糖尿病 | 关心血糖健康 |
| 降血压 | 暗示治疗高血压 | 关注血压管理 |
| 改善失眠 | 暗示治疗失眠症 | 帮助睡个好觉 |
| 增强免疫力 | 属于功效宣称 | 守护免疫力/为免疫力加油 |
| 减肥/瘦身/减脂 | 超出保健食品功能范围 | 体重管理/身材管理 |
| 预防XX | 暗示预防疾病 | 为XX健康加道防线 |
| 治疗XX | 医疗用语 | 关爱XX健康 |
| 抗癌/防癌 | 严重违规 | 不可替代，必须删除 |
| 排毒 | 夸大功效 | 帮助身体做减法 |
| 清除自由基 | 科学功效宣称 | 对抗氧化压力 |

### 效果量化（禁止）

- "3天见效""一周瘦5斤""1个月白2个度"
- "吸收率99%""纯度100%"
- "X倍吸收""X倍效果"

## 三、小红书平台敏感词

### 广告法相关

- 特价、促销、打折、折扣、优惠、限时、抢购、甩卖、清仓
- 立即购买、马上下单、赶紧抢、手慢无、错过后悔
- 年度爆款、全网最低、性价比之王

> 小红书文案中应避免直接促销话术，用"主页有同款""我吃的这款🔗"等软性引导替代

### 虚假宣传类

- 纯天然、无添加（除非有认证）
- 零副作用、安全无害
- 消字号、健字号（不得暗示批号代表疗效）
- FDA认证、欧盟认证（不得误导消费者）

### 平台限流词

- 加微信、私信我买、VX、V、wei xin
- 淘宝搜、京东搜、拼多多
- 价格相关：多少钱、价格、白菜价、划算

## 四、合规声明模板

涉及保健食品时，文案末尾必须包含以下声明：

**标准版**：
> 保健食品不是药物，不能代替药物治疗疾病

**加长版**（推荐用于科普型文案）：
> 本品为保健食品，不能代替药物治疗疾病。保健食品不具有疾病预防、治疗功能。

## 五、合规表达替换速查表

| 场景 | 禁用 | 可用 |
|------|------|------|
| 产品作用 | 治疗/改善XX | 关爱/守护/关注XX健康 |
| 效果描述 | X天见效 | 坚持一段时间后/我的感受是 |
| 免疫力 | 增强免疫力 | 守护免疫力/为免疫力加油 |
| 睡眠 | 改善失眠 | 帮助睡个好觉/关心睡眠质量 |
| 体重 | 减肥/瘦身 | 体重管理/身材管理 |
| 皮肤 | 美白/祛斑 | 肤色更匀净/皮肤状态更稳 |
| 肠道 | 治疗便秘 | 让肚子更舒服/肠道更顺畅 |
| 骨骼 | 预防骨质疏松 | 关心骨骼健康/为骨骼加油 |
| 眼睛 | 治疗近视/眼疾 | 守护用眼健康/眼睛更舒适 |
| 购买引导 | 赶紧下单/限时优惠 | 主页有同款/我吃的这款在🔗 |

## 六、自检流程

1. 搜索文案中是否包含上述违禁词
2. 检查是否有疾病名称
3. 检查是否有功效承诺和效果量化
4. 检查是否有绝对化用语
5. 检查是否有直接促销话术
6. 确认保健食品声明是否已添加
7. 全文通读，确认整体语气不像是"卖药的"
