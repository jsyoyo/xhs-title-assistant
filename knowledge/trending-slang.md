## 三、产品推荐常用句式

### 种草表达

| 阶段 | 句式 |
|------|------|
| 引出 | 后来闺蜜给我安利了 / 我妈直接寄了一瓶 / 被同事种草 |
| 使用 | 每天XX就搞定 / 撕开直接吃 / 随手一XX |
| 感受 | 最明显的感受是 / 最大的变化是 / 真的能感觉到 |
| 推荐 | 真心建议XX / 值得试一试 / 不后悔入 / 已经回购第X次 |

### 购买引导（自然植入版）

核心思路：用真实体验时间线代替硬广，让"坚持"本身就暗示囤货，让"回购"本身就引导购买。

**体验时间线写法**（推荐）：

> 我吃了第一个月的时候没什么感觉，因为一次囤了3瓶就坚持了下来，第二个月的时候感觉有些变化，第三个月时状态提升了不少

**回购暗示写法**：

> 现在第一瓶快吃完了，已经提前补了第二瓶 / 这是回购第三次了，每次都是3瓶起囤 / 吃完这瓶还会继续，已经放购物车了

**自然提及写法**：

> 我吃的这款在主页🔗 / 之前有人问我在哪买的，我放主页了 / 同款信息我置顶了

**注意**：
- 不要出现"赶紧下单""限时优惠""手慢无"等促销话术
- 不要直接说价格和折扣信息
- 购买引导最多出现一次，放在文案末尾
- 用"我的体验"替代"你该买"

## 七、Emoji 使用指南

| 场景 | 推荐emoji | 示例 |
|------|-----------|------|
| 健康/养生 | 💚🌿✨ | 守护健康✨ |
| 惊喜/推荐 | 🤫💡🔥 | 宝藏发现🤫 |
| 痛点/困扰 | 😭😣😩 | 肚子又闹了😭 |
| 变化/效果 | 🌸💫💪 | 坚持两个月💪 |
| 清单/要点 | 📌🔹❗ | 划重点📌 |
| 互动/引导 | 👇💬🔗 | 评论区见👇 |
| 保健品类 | 💊🍵🧃 | 每天两粒💊 |

### 使用原则
- 全文emoji ≤8个，不堆砌
- 每段最多1-2个
- 标题1-2个
- 避免同一emoji重复使用
- 用emoji代替标点而非额外添加
