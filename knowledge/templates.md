# 小红书保健品种草文案模板库

> 模板定义了不同类型文案的结构骨架，配合 system-prompt.md 中的写作规范使用。每次生成时选择一种模板，填充具体内容，确保结构清晰、角度多样。

---

## 模板一：种草分享型

> 最常用模板。以"我用了觉得好所以分享"的逻辑，自然种草。

### 结构

```
【钩子】痛点共鸣（1-2句）
【转折】寻找解决方案的过程
【产品】我选了XX，因为XX
【体验】使用感受+时间线（核心段落）
【利他】使用技巧/小知识（1-3条）
【引导】互动+软性购买引导
```

### 适用场景
- 单品推荐
- 自用好物分享
- "被安利后亲测"类内容

### 标题公式
- "被XX追着问的秘密🤫竟然这么简单"
- "后悔没早发现的宝藏XX"
- "XX踩坑无数次终于找到对的"

## 模板二：场景带入型

> 从具体生活场景切入，让读者代入"这不就是我吗"，然后给出解决方案。

### 结构

```
【场景】描绘一个真实的生活场景（2-3句，代入感强）
【困境】场景中的健康痛点
【发现】偶然/被推荐了解到XX
【改变】使用后的生活变化
【建议】给同类人群的建议
【引导】互动+购买引导
```

### 适用场景
- 特定人群（打工人、宝妈、学生）
- 特定时间场景（熬夜后、换季、出差）
- 特定生活状态（久坐、外卖党）

### 标题公式
- "996两年XX先扛不住了"
- "当妈以后最怕XX"
- "XX的人必看！一个习惯的改变"

---

## 模板三：干货科普型

> 先输出有价值的知识，建立信任，再自然引出产品作为解决方案之一。

### 结构

```
【疑问】提出一个常见的认知误区或问题
【科普】用通俗语言解释原理（2-4个要点）
【方案】解决这个问题的方法（1-3个）
【产品】XX是其中一种选择，我选它的理由
【自查】给读者一个自查清单
【引导】互动引导
```

### 适用场景
- 成分科普（如"为什么补钙要先补D"）
- 搭配科普（如"益生菌+益生元"）
- 认知纠偏（如"保健品不是药"）

### 标题公式
- "XX没用？可能是缺了关键一步💡"
- "90%的人都不知道的XX真相"
- "XX前必看！别再交智商税了"

## 模板四：对比测评型

> 多个产品对比或使用前后对比，用客观分析建立信任感。

### 结构

```
【背景】为什么要做这个对比
【标准】我选产品的几个维度
【对比】逐项对比（表格或要点形式）
【结论】我最终选了XX，理由是
【使用感受】使用后的真实反馈
【建议】不同人群的选择建议
【引导】互动引导
```

### 适用场景
- 同品类不同品牌对比
- 不同形态产品对比（胶囊vs粉剂vs滴剂）
- 使用前后对比

### 标题公式
- "吃了3款XX后，我选了这款"
- "XX怎么选？帮你避坑！"
- "XX对比测评，结果出乎意料"

---

## 模板五：清单合集型

> 一次推荐多个相关产品，适合做系列内容。

### 结构

```
【总览】为什么整理这个清单/我的使用背景
【清单】逐个推荐（每个3-5句：产品+理由+感受）
【总结】整体搭配建议
【利他】选购XX的通用建议
【引导】互动引导
```

### 适用场景
- "打工人必备的5款保健品"
- "换季养生清单"
- "XX人群的营养方案"

### 标题公式
- "XX必看的N个养生好物"
- "我的XX养生清单，全部亲测"
- "N款XX亲测对比，红黑榜来了"

---

## 模板六：时间线记录型

> 用时间线展示变化过程，真实感最强。

### 结构

```
【起因】为什么要开始吃XX
【第1周】初始感受（可能没什么变化，真实）
【第2-4周】逐渐出现的变化
【第1-2月】更明显的变化
【现在】整体感受+是否回购
【建议】给想试的人的建议
【引导】互动引导
```

### 适用场景
- 需要长期服用才能感受效果的保健品
- 有明显体感变化的产品

### 标题公式
- "坚持吃XX两个月，最明显的3个变化"
- "XX打卡第60天，来说真话"
- "记录吃XX的每一天，第XX天变化最大"

## 模板组合建议（批量创作时）

批量产出10篇时，建议模板分配：

| 篇数 | 模板 | 说明 |
|------|------|------|
| 3篇 | 种草分享型 | 主力模板，适合日常种草 |
| 2篇 | 场景带入型 | 覆盖不同人群场景 |
| 2篇 | 干货科普型 | 提升账号专业度和信任感 |
| 1篇 | 对比测评型 | 体现客观性，适合决策期用户 |
| 1篇 | 清单合集型 | 适合做合集内容和收藏向笔记 |
| 1篇 | 时间线记录型 | 真实感最强，适合转化犹豫用户 |

每篇的标题公式不重复，切入角度差异化，确保内容多样性。
